# Earthquake Data Analysis & Visualization

An exploratory data science project focused on processing, analyzing, and visualizing seismic activity across the globe. This project aims to identify patterns in earthquake occurrences and their geographical impact.

---

## 🚀 Project Overview

This repository provides a comprehensive workflow for handling earthquake datasets. It covers everything from data cleaning and statistical correlation to advanced visualization using Python's data science ecosystem.

## ✨ Key Features

* **Data Preprocessing:** Robust cleaning methods for handling missing values and formatting seismic timestamps.
* **Statistical Analysis:** Calculating correlations between depth, magnitude, and frequency.
* **Dynamic Visualizations:** Detailed plots including magnitude distribution, time-series analysis, and scatter plots.
* **Geographical Mapping:** Identifying high-risk zones and tectonic plate activity through coordinate plotting.

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook / VS Code
